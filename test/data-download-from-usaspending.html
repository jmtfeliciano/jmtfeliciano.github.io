Introduction to Tabular Data in R: Using usaspending.gov

Chapter 2 Data Download from USAspending.gov

2.1 How SBA assistance programs are represented in the data

The Catalog of Federal Domestic Assistance (CFDA) is a compilation of federal U.S. assistance programs. Every SBA loans and grants within USAspending.gov is labeled and tracked with their appropriate CFDA code ID and name. For example, SBA’s 7(a) loans have a CFDA code and name of 59.012 and 7(a) Loan Guarantees, respectively.

For SBA’s list of CFDA codes and names, we have compiled a list of SBA programs and how they are represented within the CFDA catalog you may access by clicking this link to our SBA box folder.

You may also search and access the CFDA code and names of every single assistance program throughout the federal government, both active and defunct, by going to the assistance listing page of sam.gov.

2.2 Warning against using older data

Data quality issues are known for older data prior to FY2018 throughout the federal government. Although the DATA Act was passed in 2013, much of the official data standards and guidance from the Treasury Department and Office of Management and Budget arrived much later. Therefore, please use caution using older data prior to FY 2018.

With this warning out, we will now go over three primary methods to download data from USAspending.gov.

2.3 Method 1: Advanced Search

Preview of the Custom Award Data page

The third method can be accessed by clicking the Advanced Search option under the Award Search section as demonstrated in the image above.

As implied by the Advanced Search title, this method includes several filters that will allow you to hone in on very specific data you might be interested in. For example, the location search filter under this method allows you to fetch awards down to city, congressional district, or zip code specific data.

This is a list of filters you could use to fetch your desired data:

  • Keyword

    • The search by Keyword option searches the following data fields: Award ID (PIID, FAIN, & URI) & Modification Number, Award Description, Award Type Names, Agency, Recipient Name, Recipient UEI, Recipient DUNS, Locations (Place of Performance & Recipient), NAICS (Code & Description), PSC (Code & Description), Business Types Names, and CFDA Program (Code & Description).

  • Time Period

    • Note: There are two options for this filter. The first is by clicking the relevant FYs. Please note that this option only goes back to FY 2008. If you need before FY 2008, you may fetch the data by inputting the dates manually using the second method. This second method allows you to request data up to FY 2001.

  • Award Type (e.g., Contracts, Grants, Loans, Other)

  • Agency

  • Treasury Account Symbols (TAS)

  • Location

  • Recipient

  • Recipient Type (e.g., Business, Minority Owned Business, Nonprofit, Individuals, etc.)

  • Award Amount

  • Assistance Listing (CFDA Program)

  • Disaster Emergency Fund Code (DEFC)

  • North American Industry Classification System (NAICS)

  • Product or Service Code (PSC)

  • Type of Contract Pricing

  • Type of Set Aside

  • Extent Competed

Two additional filters of note are (a) Assistance Listing (CFDA Program) and (b) Disaster Emergency Fund Code (DEFC).

Advanced Filter: Assistance Listing (CFDA Program)

Preview of typing CFDA code into the CFDA filter

If you are interested in data for a specific program only, the Assistance Listing (CFDA Program) filter will allow you to specify and filter for that specific program data only. Under this filter, you may start by typing the CFDA code or name of the program of interest. Please revisit the first subsection on this page for a refresher on CFDA codes or revisit the list of SBA programs and their appropriate CFDA codes from the document provided earlier.

2.4 Method 2: Award Data Archive

Preview of the Award Data Archive page

The second method can be accessed by clicking the Award Data Archive option under the Download section as demonstrated in the image above.



Preview of the simple filtering options using this method

After you scroll down, you will see a small box that will allow you to download data after specifying the desired (a) Agency, (b) Award Type (i.e., contracts or financial assistance), and (c) Fiscal Year. After picking the appropriate options and clicking the Apply button, your computer will start downloading a zipped folder containing the desired files.

Warning: Although the larger USAspending.gov platform contains data up to FY2001, this method only allows users to download data up to FY 2008.

2.5 Method 3: Custom Award Data

Preview of the Custom Award Data page

The third method can be accessed by clicking the Custom Award Option option under the Download section as demonstrated in the image above.

This option allows you to filter and download data using the following options:

  • By Award type

  • By Funding Agency

  • By Location

  • By Date Range

  • By File Format (e.g., csv, tsv, txt)

Important Note: Unlike the first method, this download method allows you to download data up to FY 2001.

According to USAspending.gov, the DEFC filter allows you to filter for spending under specific supplemental funding for major disasters and emergencies, such as the COVID-19 pandemic. For example, you may use for contracts and assistance data funded by the American Rescue Plan.

Important Note: This option will not allow you to download the requested data if the number of rows were to exceed 500,000 records. Similar to the first method, this download method only allows you to download data up to FY 2008. This method is not ideal for large batches of data as it is designed for downloading data filtered for specific parameters. To work around the 500,000 records limitation, you may need to download the desired data by chunks (e.g., by program name or code, by award type, by award amount, etc).
